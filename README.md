#  Cities Box.cpp
## 概要
完全無料の都市開発ゲーム「Cities Box」をC++で再現しています。

## 開発環境
- Windows版
- - OS : Windows 10 Home 64bit
- - Windows SDK : 10.0
- - C++ コンパイラ：C++ 17
- - 統合開発環境 : Visual Studio 2019 (v142)
- - ライブラリ：OpenSiv3D v0.4.2
- Mac版
- - OS : macOS 10.15.2 Catalina
- - C++ コンパイラ：C++ 17
- - 統合開発環境 : Xcode 10.12.1
- - ライブラリ：OpenSiv3D v0.4.2
